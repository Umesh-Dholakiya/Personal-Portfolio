It mimics the functionality of popular streaming services, with a focus on performance, scalability, and intuitive user experience.

Key Features :

✅ Search Functionality: Users can search for movies by name using the search bar at the top.
✅ Favorite List: Users can add movies to their favorites using the heart icon, which saves selections to their profile.
✅ User Authentication: Secure login & registration system to personalize user experience.
✅ Responsive Design: Fully responsive across desktop, tablet, and mobile devices.

✅ Styling: CSS, Bootstrap
